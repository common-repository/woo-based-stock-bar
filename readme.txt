== Description ==
This is simple stock bar for woocommerce which shows how much is remaining , completed from the transactions made for the products.It shows the bar in different colors from the stock available for each product quantity.

Author Email: shas2001@gmail.com
 

**Features**
 
* Easy to Install.
* Its php with css and no ajax/js.
* available in shortcode for page or post and also in widget element. 

**Notes**

*If you face any problem or find any bugs ,then please let me know.I will fix that.Please cooperate to make this plugin more better.

== Installation ==

**Normal installation**

1. Download the woocommerce-stock-bar.zip file to your computer.
1. Unzip the file.
1. Upload the `woocommerce-stock-bar` directory to your `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

That's it. You can use the the plugin by using the short code [Product_Stockbar] in your page content or you can call in custom templates php code or drop it using widget.It should be put in product detail page for best and effective display.
